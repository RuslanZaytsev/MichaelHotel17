
1. распаковать полученный архив

2. открыть терминал / оболочку ОС (bash / sh / zsh)

3. перейти в директорию папки - Test_docker_compose_image
    (вытаскиваем скачанную папку на рабочий стол, открываем терминал
    и прописываем ls, видим список доступных папок в который можно пройти,
    в списке должна быть папка Desktop, тут же в терминале пишем - cd Desktop нажимаем Enter
    дальше тут же в терминале пишем cd Test_docker_compose_image нажимаем Enter, туже в терминале
    пишем ls нажимаем Enter, в списке доступных файлов должны увидеть -
    start_App_for_Mac.txt
    start_App_for_Mac.Windows
    start_App_for_Mac.Desktop

    переходим на след пункт -> (см. пункт 4)
    )

4. в терминале ввести команду для запуска исполняемого файла
   в зависимости от вашей ОС

   а. если у вас Ubuntu, в терминале прописать - ./start_App_for_Mac.Desktop
   б. еслм у вас МакОС, в терминале прописать - ./start_App_for_Mac.txt
   в. есил у вас Windows, в терминале прописать - sh start_App_for_Windows.bat

5. заходим в брайзер на адрес - http://localhost:8085/

ВНИМАНИЕ!: если у вас ОС убунту, можно запускать файл согласно иснтрукции выше, если у вас
другая ОС, перед выполнением пунктов выше вам необходимо установить приложение
Docker Desktop c официального сайта Docker
ссылка - https://www.docker.com/
